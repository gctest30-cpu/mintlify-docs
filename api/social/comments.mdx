sidebarTitle: "Comments"

The Comments API lets you add conversation to any post on NSN. You can create a new comment under a post, retrieve all comments for a post, edit your own comments, and delete them when they are no longer needed. All four endpoints require authentication and scope their operations to a specific `user_logged` and `post` combination, ensuring that only the comment's owner can modify or remove it.

---

## Endpoints

<Accordion title="POST /{user_logged}/{post}/comment-post — Create a Comment">

Creates a new comment on the specified post, attributed to the authenticated user.

**Endpoint**

```
POST /{user_logged}/{post}/comment-post
```

**Authentication:** Required

### Path Parameters

<ParamField path="user_logged" type="string" required>
  The UUID of the authenticated user creating the comment.
</ParamField>

<ParamField path="post" type="string" required>
  The UUID of the post to comment on.
</ParamField>

### Request Body

<ParamField body="content" type="string" required>
  The text content of the comment.
</ParamField>

### Example Request

```bash
curl -X POST https://api.nsn.social/USER_ID/POST_ID/comment-post \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Great post, thanks for sharing!"
  }'
```

### Example Response

```json
{
  "message": "Comment successfully made."
}
```

### Error Responses

| Status | Description |
|--------|-------------|
| `401 Unauthorized` | Missing or invalid authentication token. |
| `404 Not Found` | The specified post does not exist. |
| `500 Internal Server Error` | An unexpected server-side error occurred. |

</Accordion>

<Accordion title="GET /{user_logged}/{post}/comments — List Comments on a Post">

Returns all comments associated with the specified post for the authenticated user's view.

**Endpoint**

```
GET /{user_logged}/{post}/comments
```

**Authentication:** Required

### Path Parameters

<ParamField path="user_logged" type="string" required>
  The UUID of the authenticated user making the request.
</ParamField>

<ParamField path="post" type="string" required>
  The UUID of the post whose comments you want to retrieve.
</ParamField>

### Example Request

```bash
curl -X GET https://api.nsn.social/USER_ID/POST_ID/comments \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

### Example Response

The endpoint returns an array of comment objects. Each object contains the comment text and its timestamps.

```json
[
  {
    "content": "Great post, thanks for sharing!",
    "created_at": "2024-06-01T10:30:00Z",
    "updated_at": "2024-06-01T10:30:00Z"
  },
  {
    "content": "I totally agree with this perspective.",
    "created_at": "2024-06-01T11:05:00Z",
    "updated_at": "2024-06-02T08:00:00Z"
  }
]
```

### Response Fields

<ResponseField name="content" type="string">
  The text body of the comment.
</ResponseField>

<ResponseField name="created_at" type="string (ISO 8601)">
  The UTC timestamp of when the comment was first created.
</ResponseField>

<ResponseField name="updated_at" type="string (ISO 8601)">
  The UTC timestamp of the most recent edit to the comment. Matches `created_at` if the comment has never been edited.
</ResponseField>

### Error Responses

| Status | Description |
|--------|-------------|
| `401 Unauthorized` | Missing or invalid authentication token. |
| `404 Not Found` | The specified post does not exist. |
| `500 Internal Server Error` | An unexpected server-side error occurred. |

</Accordion>

<Accordion title="PUT /{user_logged}/{post}/{comment}/comment-edit — Edit a Comment">

Updates the content of an existing comment. Only the user who created the comment can edit it.

**Endpoint**

```
PUT /{user_logged}/{post}/{comment}/comment-edit
```

**Authentication:** Required

### Path Parameters

<ParamField path="user_logged" type="string" required>
  The UUID of the authenticated user who owns the comment.
</ParamField>

<ParamField path="post" type="string" required>
  The UUID of the post the comment belongs to.
</ParamField>

<ParamField path="comment" type="string" required>
  The UUID of the comment to edit.
</ParamField>

### Request Body

<ParamField body="content" type="string" required>
  The updated text content to replace the existing comment body.
</ParamField>

### Example Request

```bash
curl -X PUT https://api.nsn.social/USER_ID/POST_ID/COMMENT_ID/comment-edit \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Updated: this post is even better on second read!"
  }'
```

### Example Response

```json
{
  "message": "Comment successfully edited."
}
```

### Error Responses

| Status | Description |
|--------|-------------|
| `401 Unauthorized` | Missing or invalid authentication token. |
| `403 Forbidden` | You do not own this comment. |
| `404 Not Found` | The specified post or comment does not exist. |
| `500 Internal Server Error` | An unexpected server-side error occurred. |

</Accordion>

---

<Accordion title="DELETE /{user_logged}/{post}/{comment}/comment-delete — Delete a Comment">

Removes the specified comment from the post. Only the comment's author can perform this action. Deleted comments are soft-deleted and will no longer appear in the comments list.

**Endpoint**

```
DELETE /{user_logged}/{post}/{comment}/comment-delete
```

**Authentication:** Required

### Path Parameters

<ParamField path="user_logged" type="string" required>
  The UUID of the authenticated user who owns the comment.
</ParamField>

<ParamField path="post" type="string" required>
  The UUID of the post the comment belongs to.
</ParamField>

<ParamField path="comment" type="string" required>
  The UUID of the comment to delete.
</ParamField>

### Example Request

```bash
curl -X DELETE https://api.nsn.social/USER_ID/POST_ID/COMMENT_ID/comment-delete \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

### Example Response

```json
{
  "message": "Comment successfully deleted."
}
```

### Error Responses

| Status | Description |
|--------|-------------|
| `401 Unauthorized` | Missing or invalid authentication token. |
| `403 Forbidden` | You do not own this comment. |
| `404 Not Found` | The specified post or comment does not exist. |
| `500 Internal Server Error` | An unexpected server-side error occurred. |

</Accordion>
