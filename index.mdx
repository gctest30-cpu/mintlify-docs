NSN (New Social Network) gives you a complete social networking backend over a simple REST API. Whether you are building a mobile app, a web frontend, or integrating social features into an existing product, NSN handles authentication, user profiles, posts, comments, likes, shares, and direct messaging so you can focus on your product experience.

## What you can build

NSN provides all the primitives of a social network as API endpoints:

## How it works

<Steps>
  <Step title="Create an account">
    Call `POST /signup` with your full name, username or email, password, and date of birth. You receive a JWT access token and refresh token immediately.
  </Step>
  <Step title="Authenticate your requests">
    Include the access token in the `Authorization: Bearer <token>` header on every subsequent request.
  </Step>
  <Step title="Start building">
    Create posts, search users, add comments, and explore the full API surface — all with the same token.
  </Step>
</Steps>

<Tip>
  Check the [API Reference](/api/overview) for a complete list of endpoints, request shapes, and response formats.
</Tip>
